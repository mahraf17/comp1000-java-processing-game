# Football Hopper - Java Processing Game
## Overview
Football Hopper is a mini game developed for my COMP1000 university project using the Processing programming language, which is based on Java. The project was developed throughout the semester as I learned core programming concepts step by step.

The game applies fundamental programming skills such as variables, conditionals, loops, functions, objects, movement, collision logic, scoring, and user interaction.

## Project Purpose
The purpose of this project was to demonstrate beginner-to-intermediate programming knowledge through an interactive game. Each stage of development built on weekly learning topics from the course.

## Key Features
- Player movement and interaction
- Football kicking/gameplay mechanics
- Scoreboard system
- Game restart functionality
- Custom visual design
- Basic collision and game logic

## Technologies Used
- Processing
- Java

## Files
- `M_17.pde` - Supporting Processing source file
- `Project_49041800.pde` - Main Processing project file

## What I Learned
Through this project, I improved my understanding of programming fundamentals, object-oriented thinking, game logic, debugging, and designing an interactive application from scratch.
